# WTDSight
**WTDSight** - это удобный онлайн редактор для создания пользовательских прицелов для War Thunder!

<p>
  <img src="images/logo.svg" alt="WTDSight Logo" width="150">
</p>

**Ссылка на сайт:** https://dimas70800.github.io/WTDSight/

## Что здесь есть:

- Инструмент линии
- Инструмент четырёхугольника
- Инструмент штриховки
- Инструмент векторизации (в бета-тесте, не рекомендую его использовать)
- Инструмент выделения нескольких объектов сразу
- Меню для экспорта в .blk файл с настройками прицела
- Загрузка из BLK файла
- Предпросмотр прицела

И несколько дополнительных функций
